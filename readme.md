# 🔮 Oráculo das Charadas

**Descrição:**
O Oráculo das Charadas é um sistema interativo de adivinhação que desafia os usuários a decifrarem enigmas místicos. O projeto combina charadas de uma API externa com charadas personalizadas criadas pelos próprios usuários, oferecendo uma experiência divertida e educativa. O sistema registra estatísticas de acertos e erros, permitindo que os jogadores acompanhem seu desempenho. Ideal para momentos de descontração, desenvolvimento de raciocínio lógico e diversão entre amigos.

## Funcionalidades
- **Sistema de Charadas Aleatórias**: Exibe charadas alternando entre API externa e banco local (50% chance cada)
- **Card 3D Interativo**: Clique no card para revelar a resposta do enigma
- **Registro de Estatísticas**: Contabiliza acertos, erros e total de tentativas
- **Barra de Progresso**: Visualização gráfica do percentual de acertos
- **Criador de Charadas Personalizadas**: Formulário para adicionar novas charadas ao sistema
- **Armazenamento Local**: Persistência de charadas criadas e estatísticas usando localStorage
- **Sistema de Feedback**: Notificações visuais ao acertar ou errar uma charada
- **Reset de Estatísticas**: Opção para limpar o histórico de acertos e erros

## Tecnologias Utilizadas
- **Linguagens/Frameworks:**
  - HTML5
  - CSS3 (com Glassmorphism e animações)
  - JavaScript (Vanilla)
- **API Externa:**
  - Gerador de Charadas API (https://gerador-de-charadas-api.vercel.app/)
- **Armazenamento:**
  - localStorage do navegador
- **Design:**
  - Tema místico com gradientes
  - Card 3D com efeito flip
  - Layout responsivo para mobile
